BASEL suports templates to views and controller files.

Templates for views

The standard model for views has: New button, Table to list, Search input and Form to create and update.

To create your own templates you have to use the class basel-table-head in the head of your table thead, and the class basel-table-body in the tag tbody. Look:

<table class="table table-striped" ng-init="list()">
  <thead class="basel-table-head">
  </thead>
  <tbody class="basel-table-body">
  </tbody>
</table>

And use the functions references from controller, like ng-init="list()" in the above example.

Save your template file with the name crud.html in the folder templates of your theme or app.

Templates for controllers

In the template file to controllers you have to use some key words:

CONTROLLER_NAME It will be replaced by the name of your controller.
TABLE_NAME It will be replaced by the database table name.
PRIMARY_KEY It will be replaced by the primary key of your database table.

The basic options of a BASEL controller are:

list() To list records.
save() To save records. (create and update).
cancel() To cancel edition..
edit() To open form to edit record.
delete() To delete records.

See an example

Save your template with the name controller.js in the folder templates of your theme or app.
