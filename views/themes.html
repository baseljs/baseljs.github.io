BASEL supports themes for his applications. You can create your own themes based in basel-cli.

Existing Themes

Name Description How to use
Basic App An basic theme with a top menu and Bootstrap pure. $ basel-init myApp
Tabbed app An theme multiscreen using tabs. $ basel-init myApp -b tabs

Creating Themes

You can create your own themes for BASEL. You just need to follow some parameters for the correct operation of applications created from it.

To build your new theme, you can use that as a base.

Dependencies

The basic dependencies for a basel theme are:

Dependence Version
basel-cli Latest
electron-prebuilt Latest
angular Latest
angular-route Latest

Put this in your package.json:

"dependencies": {
  "electron-prebuilt": "latest",
  "basel-cli":"latest",
  "angular": "latest",
  "angular-route": "latest"
}

If you want, you can add others dependencies like electron-prebuilt or something else.

Scripts

You need to use a NPM script in your package.json, look:

"scripts": {
   "start": "electron main.js"
}

If you want, you can add scripts to built your app using electron-prebuild or something else.

basel-cli

basel-cli is a auxiliar module to BASEL aps. It provides routes, menus, configurations from basel.json and database handle.

Usage
var basel = require('basel-cli');
var routes = basel.routes(); // getting routes {JSON Object}
var menus = basel.menu(); // getting application's menu {JSON Object}
var config = basel.config; // getting app's configuration {JSON Object}
basel.database.run("SQL Statement"); // to use database
